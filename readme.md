##About
Raytracer in c++. Unfinished, unpolished and messy but nevertheless here it is.

##Controls:
WASDQE, arrows - camera controls  
IJKLUO - light controls  
T/Y - Volumetric shadows on/off  
F/G - Random rendering tresh +/-  

##Dependencies
SFML 2.0
